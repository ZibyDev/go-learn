### Задача 5 — Своя абстракция

Объяви интерфейс `Notifier` с методом `Notify(message string) string`. Реализуй три типа уведомлений: `Email{Address string}`, `SMS{Phone string}`, `Push{DeviceID string}` — каждый возвращает свою строку формата (например, Email: `"Email на address: message"`). Напиши функцию `sendAll(notifiers []Notifier, msg string)`, которая рассылает сообщение через все каналы. Создай срез из разных нотификаторов и отправь одно сообщение.

> Это реалистичный паттерн: один интерфейс, разные реализации, единая точка отправки. Так устроены многие реальные системы.

```go
package main

import "fmt"

type Notifier interface {
	Notify(message string) string
}

type Email struct{ Address string }
type SMS struct{ Phone string }
type Push struct{ DeviceID string }

func (n Email) Notify(message string) string {
	return fmt.Sprintf("Email on address %s: %s", n.Address, message)
}
func (n SMS) Notify(message string) string {
	return fmt.Sprintf("SMS on phone %s: %s", n.Phone, message)
}
func (n Push) Notify(message string) string {
	return fmt.Sprintf("Push on device %s: %s", n.DeviceID, message)
}

func sendAll(notifiers []Notifier, msg string) {
	for _, n := range notifiers {
		fmt.Println(n.Notify(msg))
	}
}

func main() {
	n := []Notifier{Email{Address: "example@mail.com"}, SMS{Phone: "INotPhone"}, Push{DeviceID: "pc-123"}}
	sendAll(n, "Spam:)")
}
```

```bash
Push on device pc-123: Spam:)
```
