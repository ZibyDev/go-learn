### Задача 2 — Полиморфный срез

Используя `Shape` из задачи 1, создай срез `[]Shape` с несколькими разными фигурами. Напиши функцию `totalArea(shapes []Shape) float64`, суммирующую площади всех фигур. Выведи итог.

> Это демонстрирует полиморфизм: один цикл, разные реализации `Area()`.

```go
package main

import "fmt"

type Shape interface {
	Area() float64
}

type Circle struct{ Radius float64 }
type Rectangle struct{ Width, Height float64 }

func (c Circle) Area() float64    { return 3.14159 * c.Radius * c.Radius }
func (r Rectangle) Area() float64 { return r.Width * r.Height }

func totalArea(shapes []Shape) float64 {
	total := 0.0
	for _, shape := range shapes {
		total += shape.Area()
	}
	return total
}

func main() {
	shapes := []Shape{
		Circle{Radius: 3},
		Rectangle{Width: 7, Height: 4},
		Circle{Radius: 5},
		Circle{Radius: 4},
		Rectangle{Width: 11, Height: 5},
	}
	fmt.Println(totalArea(shapes))
}
```

240.0795
