### Задача 3 — Сумма чисел из разнородного среза

Напиши функцию `sumNumbers(items []any) int`, которая суммирует только элементы типа `int` из разнородного среза, игнорируя остальные. Для `[]any{1, "two", 3, true, 5}` результат должен быть `9`.

> Связка с уроком 13 (срезы) + type assertion. Подумай, какую форму assertion использовать, чтобы не паниковать на не-int.

```go
package main

import "fmt"

func sumNumbers(items []any) int {
	sum := 0
	for _, item := range items {
		if v, ok := item.(int); ok {
			sum += v
		}
	}
	return sum
}

func main() {
	fmt.Println(sumNumbers([]any{1, "two", 3, true, 5}))
}
```

```bash
9
```
