### Повторение 1 — Разворот среза + функция (уроки 7, 11, 13)

Напиши функцию `reverse(s []int)` которая разворачивает срез **на месте** (two-pointer, как в палиндроме). Поскольку срез ссылается на данные, изменение должно быть видно снаружи без указателя и без возврата. Проверь на `[]int{1, 2, 3, 4, 5}` — после вызова оригинал должен стать `[5 4 3 2 1]`.

> Связка: two-pointer (урок 7) + ссылочная природа среза (урок 13). Сравни с уроком 12, где для массива был нужен указатель `*[5]int` — здесь он не нужен, подумай почему.

```go
package main

import "fmt"

func reverseSlice(s []int) {
	n := len(s)
	for i := 0; i < n/2; i++ {
		s[i], s[n-i-1] = s[n-i-1], s[i]
	}
}

func main() {
	s := []int{1, 2, 3, 4, 5}
	fmt.Println(s)
	reverseSlice(s)
	fmt.Println(s)
}
```

Вывод:
```
[1 2 3 4 5]
[5 4 3 2 1]
```
