### Задача 2 — Настоящая копия

Создай срез `[]int{1, 2, 3}`. Сделай два «дубликата»: один через простое присваивание (`b := a`), другой через `make` + `copy`. Измени первый элемент в каждом дубликате. Покажи, что присваивание затронуло оригинал, а `copy` — нет. Объясни в комментарии разницу.

```go
package main

import "fmt"

func main() {
	s1 := []int{1, 2, 3}
	s2 := s1
	s3 := make([]int, 3)
	copy(s3, s1)
	fmt.Println(s1)
	fmt.Println(s2)
	fmt.Println(s3)
	s2[0] = 10
	s3[0] = 100
	fmt.Println(s1)
	fmt.Println(s2)
	fmt.Println(s3)
}
```

Вывод:
```
[1 2 3]
[1 2 3]
[1 2 3]
[10 2 3]
[10 2 3]
[100 2 3]
```

Разница в том, что я присвоил копии через присваивание информацию по срезу (указатель, длину и кап), а в другом случае создал новую информацию по срезу (другие указатель длина и кап), из-за этого при изменении первой копии меняется и оригинал (ссылаются на 1 место в памяти), а в другом случае изменения происходят независимо от оригинала
