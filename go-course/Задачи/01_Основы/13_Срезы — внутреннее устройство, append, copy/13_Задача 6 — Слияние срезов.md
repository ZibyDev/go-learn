### Задача 6 — Слияние срезов

Напиши функцию `merge(a, b []int) []int` которая объединяет два среза в один новый (не меняя исходные). Используй `append` с распаковкой `...`. Проверь на `[]int{1,2,3}` и `[]int{4,5,6}` → `[1 2 3 4 5 6]`.

> Подумай: если просто сделать `append(a, b...)`, не затронет ли это исходный срез `a`? Как гарантировать независимость результата?

```go
package main

import "fmt"

func merge(a, b []int) []int {
	result := make([]int, 0, len(a)+len(b))
	result = append(result, a...)
	result = append(result, b...)
	return result
}

func main() {
	s1 := []int{1, 2, 3}
	s2 := []int{4, 5, 6}
	fmt.Println(s1)
	fmt.Println(s2)
	s3 := merge(s1, s2)
	fmt.Println(s1)
	fmt.Println(s2)

	fmt.Println(s4)
}
```
