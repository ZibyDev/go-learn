### Задача 1 — len и cap

Создай срез через `make([]int, 0, 2)`. В цикле добавь в него числа от 1 до 8 через `append`, на каждом шаге печатая `len` и `cap`. Понаблюдай, в какие моменты ёмкость растёт и по какому закону (как меняется cap).

```go
package main

import "fmt"

func main() {
	s := make([]int, 0, 2)
	fmt.Println(s, len(s), cap(s))
	for i := 1; i < 10; i++ {
		s = append(s, i)
		fmt.Println(s, len(s), cap(s))
	}
}
```

Вывод:
```
[] 0 2
[1] 1 2
[1 2] 2 2
[1 2 3] 3 4
[1 2 3 4] 4 4
[1 2 3 4 5] 5 8
[1 2 3 4 5 6] 6 8
[1 2 3 4 5 6 7] 7 8
[1 2 3 4 5 6 7 8] 8 8
[1 2 3 4 5 6 7 8 9] 9 16
```

Для личного интереса добавил 9, чтобы убедиться в удваивании капа
