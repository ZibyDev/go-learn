### Задача 3 — if с инициализацией

Используя `strconv.Atoi` (возвращает `int, error`), напиши программу которая пытается преобразовать несколько строк в числа: `"42"`, `"abc"`, `"100"`. Для каждой используй `if num, err := strconv.Atoi(s); err != nil` и выведи либо число, либо сообщение об ошибке.

main.go
```go
package main

import (
	"fmt"
	"strconv"
)

func main() {
	var s1 = "42"
	var s2 = "abc"
	var s3 = "100"

	if num, err := strconv.Atoi(s1); err != nil {
		fmt.Printf("'%v' не число: %v\n", num, err)
	} else {
		fmt.Println(num)
	}
	if num, err := strconv.Atoi(s2); err != nil {
		fmt.Printf("'%v' не число: %v\n", num, err)
	} else {
		fmt.Println(num)
	}
	if num, err := strconv.Atoi(s3); err != nil {
		fmt.Printf("'%v' не число: %v\n", num, err)
	} else {
		fmt.Println(num)
	}
}
```
