### Задача 4 — Оценка по баллам

Напиши функцию `grade(score int) string` через `switch` без выражения:

- 90-100 → `"A"`
- 80-89 → `"B"`
- 70-79 → `"C"`
- 60-69 → `"D"`
- меньше 60 → `"F"`

Дополнительно: если `score < 0` или `score > 100` — верни `"Invalid"`. Проверь на 95, 83, 71, 65, 40, -5, 150.

main.go
```go
package main

import "fmt"

func grade(score int) string {
	switch {
	case score < 0 || score > 100:
		return "Invalid"
	case score >= 90:
		return "A"
	case score >= 80:
		return "B"
	case score >= 70:
		return "C"
	case score >= 60:
		return "D"
	default:
		return "F"
	}
}

func main() {
	fmt.Println(grade(95))
	fmt.Println(grade(83))
	fmt.Println(grade(71))
	fmt.Println(grade(65))
	fmt.Println(grade(40))
	fmt.Println(grade(-5))
	fmt.Println(grade(150))
}
```
