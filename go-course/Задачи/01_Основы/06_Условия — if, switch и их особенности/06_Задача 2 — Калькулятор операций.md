### Задача 2 — Калькулятор операций

Напиши функцию `calc(a, b float64, op string) (float64, bool)` которая по строке операции (`"+"`, `"-"`, `"*"`, `"/"`) через `switch` возвращает результат и `true`. Для неизвестной операции или деления на ноль — возвращает `0` и `false`.

Проверь: `calc(10, 5, "+")`, `calc(10, 0, "/")`, `calc(10, 5, "%")`.

> Вспомни из урока 4: деление на ноль для float не паникует, но логически его лучше отсечь явно.

main.go
```go
package main

import "fmt"

func calc(a, b float64, op string) (float64, bool) {
	switch {
	case op == "+":
		return a + b, true
	case op == "-":
		return a - b, true
	case op == "*":
		return a * b, true
	case op == "/":
		if b == 0 {
			return 0, false
		}
		return a / b, true
	default:
		return 0, false
	}
}

func main() {
	fmt.Println(calc(10, 5, "+"))
	fmt.Println(calc(10, 0, "/"))
	fmt.Println(calc(10, 5, "%"))
}
```
