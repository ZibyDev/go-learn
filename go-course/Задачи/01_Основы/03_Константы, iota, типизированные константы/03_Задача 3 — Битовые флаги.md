### Задача 3 — Битовые флаги

Создай тип `Ability uint8` с флагами способностей персонажа через `1 << iota`:

```
CanFly
CanSwim
CanClimb
CanDig
```

Напиши программу которая:

1. Создаёт персонажа со способностями `CanFly` и `CanSwim`
2. Проверяет наличие каждой способности и выводит результат
3. Добавляет способность `CanClimb`
4. Убирает способность `CanFly`

main.go
```go
package main

import "fmt"

type Ability uint8

const (
	CanFly = 1 << iota
	CanSwim
	CanClimb
	CanDig
)

func main() {
	character := CanFly | CanSwim

	fmt.Printf("Can fly: %t\n", character&CanFly != 0)
	fmt.Printf("Can swim: %t\n", character&CanSwim != 0)
	fmt.Printf("Can climb: %t\n", character&CanClimb != 0)
	fmt.Printf("Can dig: %t\n", character&CanDig != 0)

	character |= CanClimb
	fmt.Printf("After learning to climb:\n")
	fmt.Printf("Can fly: %t\n", character&CanFly != 0)
	fmt.Printf("Can swim: %t\n", character&CanSwim != 0)
	fmt.Printf("Can climb: %t\n", character&CanClimb != 0)
	fmt.Printf("Can dig: %t\n", character&CanDig != 0)

	character &^= CanFly
	fmt.Printf("After forgetting how to swim:\n")
	fmt.Printf("Can fly: %t\n", character&CanFly != 0)
	fmt.Printf("Can swim: %t\n", character&CanSwim != 0)
	fmt.Printf("Can climb: %t\n", character&CanClimb != 0)
	fmt.Printf("Can dig: %t\n", character&CanDig != 0)
}
```
