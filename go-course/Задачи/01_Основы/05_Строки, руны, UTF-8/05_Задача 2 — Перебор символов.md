### Задача 2 — Перебор символов

Возьми строку `"Hi Мир"`. Перебери её через `for ... range` и для каждого символа выведи: байтовую позицию (index), сам символ (`%c`) и его код руны (`%d`). Обрати внимание как "прыгает" index на кириллических символах.

> Это единственный новый синтаксис в задачах — `for index, r := range s`. Циклы подробно будут в уроке 7, здесь просто используй эту форму как показано в примере 2.

main.go
```go
package main

import (
	"fmt"
)

func main() {
	str := "Hi Мир"
	for index, r := range str {
		fmt.Printf("Index: %v, %c (код руны: %d)\n", index, r, r)
	}
}
```
