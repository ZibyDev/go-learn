### Задача 4 — string ловушка

Возьми число `72`. Выведи два варианта его преобразования в строку:

- через `string(rune(72))`
- через `strconv.Itoa(72)`

Покажи разницу в выводе и объясни в комментарии что произошло в первом случае.

main.go
```go
package main

import (
	"fmt"
	"strconv"
)

func main() {
	var x = 72
	fmt.Println(string(rune(x)))
	fmt.Println(strconv.Itoa(72))
}
```

1 метод выводит символ, которому соответствует значение байта 72
2 метод выводит число 72 как строку
