### Задача 3 — Разворот строки

Напиши функцию `reverse(s string) string`, которая разворачивает строку задом наперёд. Проверь на `"Hello"` → `"olleH"` и на `"Привет"` → `"тевирП"`.

> Подсказка: преврати строку в `[]rune`, поменяй элементы местами, конвертируй обратно в `string`. Почему именно `[]rune`, а не `[]byte`? Подумай что будет с кириллицей если разворачивать по байтам. Для обмена местами можно обойтись без цикла, если строки короткие — но если знаешь как сделать циклом `for`, попробуй (форма `for i := 0; i < n; i++` работает как в C).

main.go
```go
package main

import (
	"fmt"
	"unicode/utf8"
)

func reverse(s string) string {
	var strRune = []rune(s)
	var n = utf8.RuneCountInString(s)
	for i := 0; i < n/2; i++ {
		strRune[i], strRune[n-i-1] = strRune[n-i-1], strRune[i]
	}
	return string(strRune)
}

func main() {
	var str1 = "Hello"
	var str2 = "Привет"
	fmt.Println(str1 + " " + reverse(str1))
	fmt.Println(str2 + " " + reverse(str2))
}
```
