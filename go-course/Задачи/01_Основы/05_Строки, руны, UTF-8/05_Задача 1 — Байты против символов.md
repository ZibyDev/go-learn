### Задача 1 — Байты против символов

Объяви строку, содержащую и латиницу, и кириллицу (например `"Go Привет"`). Выведи:

- её длину в байтах (`len`)
- её длину в символах (`utf8.RuneCountInString`)

Объясни в комментарии почему числа разные.

main.go
```go
package main

import (
	"fmt"
	"unicode/utf8"
)

func main() {
	str := "Go Привет"
	fmt.Println(len(str))
	fmt.Println(utf8.RuneCountInString(str))
}
```
Первое число - 15, выводит количество байт, первые три символа по 1 байту (ASCII), слово "Привет" по 2 байта (правила UTF-8)
Второе число (9) показывает просто длину строки по символам (сначала конвертирует символы в руны, а потом считает их количество)
