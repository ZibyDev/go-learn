### Задача 4 — Игровые переменные

Создай набор переменных для описания игрового персонажа:

```
Имя: "Hero"
Уровень: 1
Здоровье: 100.0
Мана: 50.0
Жив: true
Координаты X, Y: 0, 0
```

Выведи всё через один `fmt.Printf` с форматированием.

main.go
```go
package main

import "fmt"

func main() {
	name := "Hero"
	lvl := 1
	health := 100
	mana := 50
	alive := true
	x, y := 0, 0
	fmt.Printf("Имя: %s\nУровень: %d\nЗдоровье: %d\nМана: %d\nЖив: %t\nКоординаты: (%d, %d)\n", name, lvl, health, mana, alive, x, y)
}
```
