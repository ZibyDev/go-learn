### Задача 2 — Типы и приведение

Напиши программу которая:

1. Объявляет `width := 1920` и `height := 1080` как `int`
2. Вычисляет соотношение сторон как `float64` (должно получиться `1.7777...`)
3. Вычисляет площадь экрана и выводит результат

> Подсказка: для деления нужно явное приведение типов

main.go
```go
package main

import "fmt"

func main() {
	width := 1920
	height := 1080
	fmt.Println(float64(width) / float64(height))
	fmt.Println(width * height)
}
```
