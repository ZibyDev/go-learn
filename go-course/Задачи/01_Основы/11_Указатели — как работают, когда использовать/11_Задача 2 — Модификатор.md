### Задача 2 — Модификатор

Напиши функцию `applyDamage(health *int, damage int)` которая вычитает урон из здоровья по указателю, но не даёт здоровью уйти ниже нуля (если урон больше здоровья — ставь 0). Проверь на `health=100, damage=30` и `health=20, damage=50`.

```go
package main

import "fmt"

func applyDamage(health *int, damage int) {
	if damage > *health {
		*health = 0
	} else {
		*health -= damage
	}
}

func main() {
	health := 100
	fmt.Println(health)
	applyDamage(&health, 30)
	fmt.Println(health)
	applyDamage(&health, 50)
	fmt.Println(health)
	applyDamage(&health, 50)
	fmt.Println(health)
}
```
