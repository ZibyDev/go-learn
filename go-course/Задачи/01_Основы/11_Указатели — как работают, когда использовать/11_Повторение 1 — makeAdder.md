### Повторение 1 — Указатели + замыкания (уроки 10, 11)

Напиши функцию `makeAdder(start int) (func(int), func() int)` которая возвращает две функции, работающие с **общим состоянием через указатель**. Первая прибавляет к значению, вторая возвращает текущее. Реализуй через захват указателя на переменную, а не самой переменной (чтобы потренировать оба механизма). Сравни с тем, как ты делал калькулятор в уроке 10 через прямой захват.

> Связка: замыкания (урок 10) + указатели (урок 11). Подумай, в чём разница между захватом переменной напрямую и захватом указателя на неё — для этой задачи результат одинаковый, но механизм разный.

```go
package main

import "fmt"

func makeAdder(start *int) (func(int), func() int) {
	add := func(x int) {
		*start = *start + x
	}
	result := func() int {
		return *start
	}
	return add, result
}

func main() {
	s := 0
	add, res := makeAdder(&s)
	add(5)
	fmt.Println(res())
	add(10)
	fmt.Println(res())
	add(15)
	fmt.Println(res())
}
```

Если на прошлом уроке я в функции создавал переменную, то в этом ссылаюсь на переменную из функции main
