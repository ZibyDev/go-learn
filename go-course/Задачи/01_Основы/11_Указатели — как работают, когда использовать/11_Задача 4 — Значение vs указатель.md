### Задача 4 — Значение vs указатель

Напиши две функции: `tryDouble(x int)` (принимает по значению) и `doublePtr(x *int)` (по указателю). Обе пытаются удвоить аргумент. В `main` вызови обе на одной переменной и покажи, что меняет оригинал только версия с указателем. Объясни в комментарии почему.

```go
package main

import "fmt"

func tryDouble(x int) {
	x *= 2
	fmt.Println("Удвоение без указателя:", x)
}

func doublePtr(x *int) {
	*x *= 2
	fmt.Println("Удвоение с указателем", *x)
}

func main() {
	x := 2
	fmt.Println("Первоначальное значение до функций:", x)
	tryDouble(x)
	fmt.Println("Первоначальное значение после tryDouble:", x)
	doublePtr(&x)
	fmt.Println("Первоначальное значение после doublePtr:", x)
}
```

1 функция получает значение и не может изменить значение той переменной, которую мы положили в функцию, когда как 2 функция работает с первоначальной переменной и может менять её в main
