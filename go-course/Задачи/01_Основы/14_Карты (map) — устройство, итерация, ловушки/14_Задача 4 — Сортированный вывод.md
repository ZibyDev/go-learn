### Задача 4 — Сортированный вывод

Создай карту `map[string]int` с несколькими парами. Выведи её содержимое в **отсортированном по ключам** порядке. Собери ключи в срез, отсортируй через `sort.Strings`, затем пройди по отсортированным ключам.

> Связка: срезы (урок 13) + карты (урок 14) + случайность порядка итерации. Пакет `sort` подробно в будущем, здесь используй `sort.Strings(keys)` как готовый инструмент.

```go
package main

import (
	"fmt"
	"sort"
)

func main() {
	m := map[string]int{
		"a": 4,
		"D": 2,
		"r": 1,
		"M": 5,
		"A": 3,
	}
	s := []string{}
	for k := range m {
		s = append(s, k)
	}
	sort.Strings(s)
	for _, i := range s {
		fmt.Printf("%s: %d\n", i, m[i])
	}
}
```

```
A: 3
D: 2
M: 5
a: 4
r: 1
```
