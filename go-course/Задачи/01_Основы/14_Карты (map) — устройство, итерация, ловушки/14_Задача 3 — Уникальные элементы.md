### Задача 3 — Уникальные элементы

Напиши функцию `unique(nums []int) []int` которая возвращает срез уникальных элементов в порядке первого появления. Используй карту-множество (`map[int]bool`) для отслеживания виденного. Проверь на `[]int{1, 2, 2, 3, 1, 4, 3}` → `[1 2 3 4]`.

```go
package main

import "fmt"

func unique(nums []int) []int {
	check := map[int]bool{}
	result := []int{}
	for _, i := range nums {
		if !check[i] {
			check[i] = true
			result = append(result, i)
		}
	}
	return result
}

func main() {
	slice := []int{1, 2, 2, 3, 1, 4, 3}
	fmt.Println(unique(slice))
}
```

```
[1 2 3 4]
```
