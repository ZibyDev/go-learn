### Задача 1 — Частота слов

Напиши функцию `wordFrequency(words []string) map[string]int` которая считает, сколько раз встречается каждое слово. Для `["a", "b", "a", "c", "a", "b"]` результат: `a→3, b→2, c→1`. Используй приём `m[word]++`.

```go
package main

import "fmt"

func wordFrequency(words []string) map[string]int {
	result := map[string]int{}
	for _, i := range words {
		result[i]++
	}
	return result
}

func main() {
	slice := []string{"a", "b", "a", "c", "a", "b"}
	fmt.Println(wordFrequency(slice))
}
```

```
map[a:3 b:2 c:1]
```
