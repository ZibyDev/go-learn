### Повторение 1 — Группировка через карту (уроки 5, 13, 14)

Напиши функцию `groupByLength(words []string) map[int][]string` которая группирует слова по их длине (в рунах!). Ключ — длина слова, значение — срез слов этой длины. Для `["go", "cat", "do", "dog", "a"]` результат: `1→[a]`, `2→[go, do]`, `3→[cat, dog]`.

> Связка: длина в рунах через `utf8.RuneCountInString` (урок 5) + карта значений-срезов с `append` (уроки 13, 14). Вспомни: `append(m[key], value)` работает даже для отсутствующего ключа.

```go
package main

import (
	"fmt"
	"unicode/utf8"
)

func groupByLength(words []string) map[int][]string {
	result := map[int][]string{}
	for _, i := range words {
		n := utf8.RuneCountInString(i)
		result[n] = append(result[n], i)
	}
	return result
}

func main() {
	slice := []string{"go", "cat", "do", "dog", "a"}
	fmt.Println(groupByLength(slice))
}
```

```
map[1:[a] 2:[go do] 3:[cat dog]]
```
