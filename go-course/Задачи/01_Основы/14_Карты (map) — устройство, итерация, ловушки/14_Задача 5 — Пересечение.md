### Задача 5 — Пересечение

Напиши функцию `intersection(a, b []int) []int` которая возвращает элементы, присутствующие в **обоих** срезах (без дубликатов). Используй карту: сначала запиши все элементы `a` в множество, потом пройди по `b` и собери те, что есть в множестве. Проверь на `[]int{1,2,3,4}` и `[]int{3,4,5,6}` → `[3 4]`.

> Подумай, как избежать дубликатов в результате, если в `b` элемент встречается дважды.

```go
package main

import "fmt"

func intersection(a, b []int) []int {
	inA := map[int]bool{}
	for _, x := range a {
		inA[x] = true
	}
	result := []int{}
	added := map[int]bool{}
	for _, x := range b {
		if !added[x] && inA[x] {
			result = append(result, x)
			added[x] = true
		}
	}
	return result
}

func main() {
	s1 := []int{1, 2, 3, 4}
	s2 := []int{3, 4, 5, 6}
	fmt.Println(intersection(s1, s2))
}
```

```
[3 4]
```
