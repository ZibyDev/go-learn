Создай проект со следующей структурой:

```
task2/
├── go.mod
├── main.go
└── info.go
```

В `info.go` напиши функцию `printInfo()` которая выводит любое сообщение. В `main.go` вызови эту функцию.

main.go
```go
package main

func main() {
	printInfo()
}

```

info.go
```go
package main

import "fmt"

func printInfo() {
	fmt.Printf("Message to complete task 2")
}

```
