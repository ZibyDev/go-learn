### Задача 2 — Числа Фибоначчи

Напиши функцию `fib(n int) int` которая возвращает n-е число Фибоначчи через цикл (последовательность: 0, 1, 1, 2, 3, 5, 8...). `fib(0)=0`, `fib(1)=1`, `fib(7)=13`.

> Подсказка: храни два предыдущих значения в переменных и обновляй их. Вспомни множественное присваивание из урока 2: `a, b = b, a+b`.

main.go
```go
package main

import "fmt"

func fib(n int) int {
	var a, b = 0, 1
	if n == 0 {
		return a
	}
	for i := 2; i <= n; i++ {
		a, b = b, a+b
	}
	return b
}

func main() {
	fmt.Println(fib(0))
	fmt.Println(fib(1))
	fmt.Println(fib(7))
}
```
