### Задача 1 — Факториал

Напиши функцию `factorial(n int) int` которая вычисляет n! через цикл (не рекурсию). `factorial(5)` = 120, `factorial(0)` = 1.

> Подумай про граничный случай: `0! = 1` по определению.

main.go
```go
package main

import "fmt"

func factorial(n int) int {
	var res = 1
	for i := 2; i <= n; i++ {
		res *= i
	}
	return res
}

func main() {
	fmt.Println(factorial(5))
}
```
