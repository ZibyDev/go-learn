### Задача 3 — Палиндром

Напиши функцию `isPalindrome(s string) bool` которая проверяет, читается ли строка одинаково в обе стороны. Используй два индекса (с начала и с конца), двигай их навстречу. Проверь на `"level"` → true, `"hello"` → false, `"Привет"` → false.

> Связка с уроком 5: преврати в `[]rune`, чтобы корректно работать с кириллицей. Это похоже на разворот строки, но без создания новой строки — просто сравнивай.

main.go
```go
package main

import "fmt"

func isPalindrome(s string) bool {
	var strRune = []rune(s)
	var n = len(strRune)
	for i := 0; i < n/2; i++ {
		if strRune[i] != strRune[n-i-1] {
			return false
		}
	}
	return true
}

func main() {
	fmt.Println(isPalindrome("level"))
	fmt.Println(isPalindrome("hello"))
	fmt.Println(isPalindrome("Привет"))
}
```
