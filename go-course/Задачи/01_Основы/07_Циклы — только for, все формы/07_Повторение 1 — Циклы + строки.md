### Повторение 1 — Циклы + строки (урок 5 + 7)

Напиши функцию `countVowels(s string) int` которая считает количество гласных (a, e, i, o, u — латинские) в строке. Используй `for range` по строке и `switch` или `if` для проверки. Проверь на `"Hello World"` (должно быть 3).

> Связка: перебор рун из урока 5 + циклы из урока 7 + условия из урока 6.

main.go
```go
package main

import "fmt"

func countVowels(s string) int {
	var countVow = 0
	for _, r := range s {
		switch r {
		case 'a', 'e', 'i', 'o', 'u':
			countVow++
		}
	}
	return countVow
}

func main() {
	fmt.Println(countVowels("Hello World"))
}
```
