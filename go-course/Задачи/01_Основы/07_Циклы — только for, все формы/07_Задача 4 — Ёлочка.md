### Задача 4 — Ёлочка

Напиши функцию `tree(height int)` которая печатает «ёлочку» из звёздочек. Для `height = 4`:

```
   *
  ***
 *****
*******
```

> Подсказка: используй вложенные циклы — внешний по строкам, внутренние для пробелов перед звёздочками и для самих звёздочек. На строке `i` (с 0): `height-i-1` пробелов и `2*i+1` звёздочек.

main.go
```go
package main

import "fmt"

func tree(height int) {
	for i := range height {
		for j := 0; j < height-i-1; j++ {
			fmt.Print(" ")
		}
		for j := 0; j < 2*i+1; j++ {
			fmt.Print("*")
		}
		fmt.Println()
	}
}

func main() {
	tree(4)
	tree(7)
	tree(2)
}
```
