### Задача 6 — Поверхностная копия

Объяви структуру `Inventory{Items []string}`. Создай экземпляр, скопируй его в другую переменную через присваивание (`b := a`). Измени элемент среза `Items` в копии. Покажи, что изменение затронуло **оба** экземпляра, и объясни почему (поверхностное копирование + ссылочная природа среза). Затем покажи, как сделать глубокую копию через `make`+`copy`.

> Связка: структуры (урок 15) + ссылочная природа срезов (урок 13). Это важнейшая тонкость — копия структуры не копирует данные срезов внутри.

```go
package main

import "fmt"

type Inventory struct {
	Items []string
}

func main() {
	inv1 := Inventory{Items: []string{"Меч", "Щит", "Топор"}}
	inv2 := inv1
	inv2.Items[1] = "Зелье"
	fmt.Println(inv2)
	fmt.Println(inv1)

	inv3 := Inventory{Items: make([]string, len(inv1.Items))}
	copy(inv3.Items, inv1.Items)
	inv3.Items[1] = "Щит"
	fmt.Println(inv3)
	fmt.Println(inv1)
}
```
Вывод:
```
{[Меч Зелье Топор]}
{[Меч Зелье Топор]}
{[Меч Щит Топор]}
{[Меч Зелье Топор]}
```
До make используется простое копирование структуры, то есть копируются все значения внутри неё, а значит копируются параметры среза, а не его внутренности
