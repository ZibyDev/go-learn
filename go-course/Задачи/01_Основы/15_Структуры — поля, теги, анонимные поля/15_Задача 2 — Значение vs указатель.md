### Задача 2 — Значение vs указатель

Объяви структуру `Counter{Value int}`. Напиши `incByValue(c Counter)` и `incByPointer(c *Counter)`. Покажи, что только версия с указателем меняет оригинал. Объясни в комментарии почему (связь с уроком 11).

```go
package main

import "fmt"

type Counter struct {
	Value int
}

func incByValue(c Counter) {
	c.Value++
}

func incByPointer(c *Counter) {
	c.Value++
}

func main() {
	c := Counter{Value: 1}
	fmt.Println(c)
	incByValue(c)
	fmt.Println(c)
	incByPointer(&c)
	fmt.Println(c)
}
```
Вывод:
```
{1}
{1}
{2}
```
Комментарии: 
Так происходит, потому что структура как массив копируется в функцию, то есть в функции полная копия элемента, который мы хотим поменять. Когда мы указываем на структуру, мы не создаем копию, а только указатель, функция уже работает с ячейками памяти, на которые указывает указатель
