### Задача 4 — Встраивание

Объяви структуру `Base{ID int; Name string}` и структуру `User`, которая **встраивает** `Base` и добавляет поле `Email string`. Создай `User`, обратись к полям `ID` и `Name` напрямую (без указания `Base`). Покажи, что промоушен полей работает.

```go
package main

import "fmt"

type Base struct {
	ID   int
	Name string
}

type User struct {
	Base
	Email string
}

func main() {
	u1 := User{Base{ID: 1, Name: "John"}, "johm@mail.com"}
	fmt.Println(u1.ID)
	fmt.Println(u1.Name)
	fmt.Println(u1.Email)
}
```
Вывод:
```
1
John
johm@mail.com
```
