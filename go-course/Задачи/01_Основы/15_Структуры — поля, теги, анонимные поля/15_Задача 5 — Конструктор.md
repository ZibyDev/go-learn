### Задача 5 — Конструктор

Объяви структуру `Timer{seconds int}`. Напиши конструктор `NewTimer(minutes int) *Timer`, который принимает минуты и сохраняет их как секунды (умножив на 60). Используй идиому `NewX`, возвращающую указатель. Проверь, создав таймер на 2 минуты (должно быть 120 секунд).

```go
package main

import "fmt"

type Timer struct {
	seconds int
}

func NewTimer(minutes int) *Timer {
	return &Timer{seconds: minutes * 60}
}

func main() {
	t1 := NewTimer(2)
	t2 := NewTimer(5)
	fmt.Printf("%+v\n", t1)
	fmt.Printf("%+v\n", t2)
}
```
Вывод:
```
&{seconds:120}
&{seconds:300}
```
