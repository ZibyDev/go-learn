### Задача 2 — Безопасное сложение

Напиши функцию `safeAddUint8(a, b uint8) (uint8, bool)` которая:

- возвращает сумму и `true` если переполнения нет
- возвращает `0` и `false` если результат превысит `math.MaxUint8`

Протестируй на парах `(100, 50)` → ok и `(200, 100)` → переполнение.

> Подсказка: проверяй ДО сложения, иначе переполнение уже произойдёт.

main.go
```go
package main

import "fmt"

func safeAddUnit8(a, b uint8) (uint8, bool) {
	if a > 255-b {
		return 0, false
	} else {
		return a + b, true
	}
}

func main() {
	fmt.Println(safeAddUnit8(100, 50))
	fmt.Println(safeAddUnit8(200, 100))
}
```
