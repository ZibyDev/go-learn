### Задача 1 — Поймать переполнение

Возьми `uint8` со значением `250`. В цикле прибавляй по `1` десять раз, каждый раз выводя значение. Понаблюдай момент переполнения (когда 255 → 0). Используй `math.MaxUint8` в коде вместо магического числа `255`.

main.go
```go
package main

import "fmt"

func main() {
	var overflowVar uint8 = 250
	var cnt = 0
	for cnt < 10 {
		fmt.Println(overflowVar)
		overflowVar++
		cnt++
	}
	fmt.Println(overflowVar)
}
```
