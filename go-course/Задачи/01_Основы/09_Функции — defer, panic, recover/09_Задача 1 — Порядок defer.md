### Задача 1 — Порядок defer

Напиши функцию которая в цикле от 1 до 5 объявляет `defer fmt.Println(i)`, а в конце печатает `"конец функции"`. Перед запуском **предскажи** на бумаге что выведется, потом проверь. Объясни в комментарии почему такой порядок и почему значения именно такие.

main.go
```go
package main

import "fmt"

func main() {
	for i := 1; i < 6; i++ {
		defer fmt.Println(i)
	}
	fmt.Println("конец функции")
}
```

Предсказание: Так как defer работает по принципу стека, то сначала выполнится все, что идет в коде (вывод "конец функции"), потом с последнего дефера по первый.

Так и оказалось
При этом если обернуть цикл в defer func(), то вывод будет идти в порядке цикла
