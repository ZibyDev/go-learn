### Задача 3 — Безопасный вызов

Напиши функцию `safeAccess(arr []int, index int) (value int, err error)` которая обращается к элементу среза по индексу. Если индекс выходит за границы — обращение вызовет панику; перехвати её через `defer`+`recover` и верни как ошибку `fmt.Errorf("индекс %d вне границ", index)`. Проверь на корректном индексе и на заведомо большом.

> Срезы подробно в уроке 13, здесь просто используй переданный `arr []int` и обращение `arr[index]`.

main.go
```go
package main

import "fmt"

func safeAccess(arr []int, index int) (value int, err error) {
	defer func() {
		if r := recover(); r != nil {
			err = fmt.Errorf("индекс %d вне границ", index)
		}
	}()
	return arr[index], nil
}

func main() {
	var arr = []int{1, 2, 3, 4, 5}
	fmt.Println(safeAccess(arr, 0))
	fmt.Println(safeAccess(arr, 4))
	fmt.Println(safeAccess(arr, 5))
	fmt.Println(safeAccess(arr, -1))
}
```
