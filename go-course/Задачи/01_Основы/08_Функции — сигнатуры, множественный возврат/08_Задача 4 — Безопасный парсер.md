### Задача 4 — Безопасный парсер

Напиши функцию `parsePositive(s string) (int, error)` которая:

- парсит строку в число через `strconv.Atoi`
- если парсинг не удался — возвращает ошибку как есть
- если число отрицательное — возвращает свою ошибку через `fmt.Errorf("число должно быть положительным, получено %d", n)`
- иначе возвращает число и `nil`

Проверь на `"42"`, `"-5"`, `"abc"`. Обработай результат через `if ...; err != nil`.

main.go
```go
package main

import (
	"fmt"
	"strconv"
)

func parsePositive(s string) (int, error) {
	if num, err := strconv.Atoi(s); err != nil {
		return 0, err
	} else if num < 0 {
		return 0, fmt.Errorf("число должно быть положительным, получено %d", num)
	} else {
		return num, nil
	}
}

func main() {
	fmt.Println(parsePositive("42"))
	fmt.Println(parsePositive("-5"))
	fmt.Println(parsePositive("abc"))
}
```
