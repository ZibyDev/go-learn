### Задача 3 — Калькулятор-замыкание

Напиши функцию `calculator() (add func(int), result func() int)` которая возвращает **две** функции: одна добавляет число к внутренней сумме, другая возвращает текущую сумму. Обе должны разделять одну захваченную переменную `sum`.

```
add(10); add(5); result() → 15
```

> Подсказка: объяви `sum` внутри `calculator`, и обе возвращаемые функции будут её захватывать. Это покажет, что замыкание разделяется между несколькими функциями.

main.go
```go
package main

import "fmt"

func calculator() (add func(int), result func() int) {
	sum := 0
	add = func(x int) {
		sum += x
	}
	result = func() int {
		return sum
	}
	return
}

func main() {
	add, result := calculator()
	add(10)
	fmt.Println(result())
	add(5)
	fmt.Println(result())
	add(35)
	fmt.Println(result())
}
```
