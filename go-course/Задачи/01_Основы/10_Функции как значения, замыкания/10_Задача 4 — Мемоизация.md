### Задача 4 — Мемоизация

Напиши функцию `memoizedSquare() func(int) int` которая возвращает функцию вычисления квадрата, но **кэширует** уже вычисленные результаты в карте `map[int]int` (карты базово; подробно урок 14). При повторном запросе того же числа — берёт из кэша. Добавь `fmt.Println("вычисляю", n)` чтобы видеть, когда происходит реальное вычисление, а когда берётся из кэша.

> Подсказка: захвати `cache := map[int]int{}` в замыкании. Проверяй наличие через `v, ok := cache[n]`. Карты разберём подробно в уроке 14, здесь используй базово.

main.go
```go
package main

import "fmt"

func memoizedSquare() func(int) int {
	cache := map[int]int{}
	return func(n int) int {
		v, ok := cache[n]
		if ok {
			return v
		}
		fmt.Println("вычисляю", n)
		res := n * n
		cache[n] = res
		return res
	}
}

func main() {
	f := memoizedSquare()
	fmt.Println(f(2))
	fmt.Println(f(4))
	fmt.Println(f(2))
	fmt.Println(f(3))
	fmt.Println(f(7))
	fmt.Println(f(4))
}
```
