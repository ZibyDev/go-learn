### Задача 1 — Счётчик

Напиши функцию `makeCounter() func() int` которая возвращает замыкание-счётчик. Создай два независимых счётчика, вызови каждый несколько раз и убедись, что они не влияют друг на друга.

main.go
```go
package main

import "fmt"

func makeCounter() func() int {
	count := 0
	return func() int {
		count++
		return count
	}
}

func main() {
	counter1 := makeCounter()
	counter2 := makeCounter()

	fmt.Println(counter1())
	fmt.Println(counter1())
	fmt.Println(counter2())
	fmt.Println(counter1())
	fmt.Println(counter2())
	fmt.Println(counter1())
}
```

Убедился, вывод:
```
1
2
1
3
2
4
```
