### Задача 3 — Сравнение

Создай три массива `[3]int`: два одинаковых и один отличающийся. Сравни их через `==` и выведи результаты. Объясни в комментарии, почему для массивов это работает, а для срезов — нет.

```go
package main

import "fmt"

func main() {
	arr1 := [3]int{1, 2, 3}
	arr2 := [3]int{1, 2, 3}
	arr3 := [3]int{3, 2, 1}

	fmt.Println(arr1 == arr2)
	fmt.Println(arr1 == arr3)
}
```
Вывод:
```
true
false
```
Сравнение массивов возможно только потому, что массив это несколько элементов 1 типа, как правило они расположены в памяти подряд (то есть занимают место в памяти вместе), когда как срезы это динамический массив, элементы которого могут находится в разных ячейках памяти и сравнивать их будет долго (нужно для каждого элемента запрашивать значение)
