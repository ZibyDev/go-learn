### Задача 1 — Копия или ссылка

Создай массив `[4]int`, присвой его другой переменной, измени один элемент в копии. Выведи оба массива и убедись, что оригинал не изменился. Затем сделай то же с передачей в функцию. Объясни в комментарии поведение.

```go
package main

import "fmt"

func modify(arr [4]int) {
	arr[2] = 1234
}

func main() {
	a := [4]int{1, 2, 3, 4}
	b := a
	b[2] = 1234
	fmt.Println(a)
	fmt.Println(b)
	modify(a)
	fmt.Println(a)
}
```
Вывод:
```
[1 2 3 4]
[1 2 1234 4]
[1 2 3 4]
```
Все просто, не изменяется 1 массив, потому что массив 2 его копия, которая лежит отдельно в памяти
