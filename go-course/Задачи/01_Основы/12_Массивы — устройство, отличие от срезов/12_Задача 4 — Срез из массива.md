### Задача 4 — Срез из массива

Создай массив `[5]int{10, 20, 30, 40, 50}`. Получи из него срез элементов с индексами 1-3 (через `arr[1:4]`). Измени первый элемент среза и покажи, что **изменился исходный массив**. Объясни почему.

> Это подводка к уроку 13: срез ссылается на массив, а не копирует его.

```go
package main

import "fmt"

func main() {
	arr := [5]int{10, 20, 30, 40, 50}
	slice := arr[1:4]
	slice[0] = 100
	fmt.Println(arr)
	fmt.Println(slice)
}

```
Вывод:
```
[10 100 30 40 50]
[100 30 40]
```
Как в комментарии к прошлой задаче я говорил, что срез ссылается на элементы (то есть по факту динамический массив указателей), из-за этого исходный массив изменился, когда я изменил элемент среза
